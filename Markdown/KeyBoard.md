#### 键盘代码说明

CH9328采用标准USB键盘，标准USB键盘包括8Byte

| Offset | 描述                       |
| ------ | -------------------------- |
| 0      | Modifier keys status       |
| 1      | Reserved field（保留字段） |
| 2      | Keypress #1                |
| 3      | Keypress #2                |
| 4      | Keypress #3                |
| 5      | Keypress #4                |
| 6      | Keypress #5                |
| 7      | Keypress #6                |
